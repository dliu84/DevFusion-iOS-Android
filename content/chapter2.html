Class 2 - Swift Basics

Introduction to Swift

Swift is a powerful and intuitive programming language developed by Apple Inc. for iOS, macOS, watchOS, and tvOS app development. Introduced in 2014, it is designed to work with Apple's Cocoa and Cocoa Touch frameworks and the large body of existing Objective-C code written for Apple products.

Why Choose Swift?

1. Safe & Easy to Read: Swift's syntax and language constructions eliminate the types of unsafe code classes often found in C and Objective-C.

2. Fast & Powerful: Swift code is performance-optimized and can compete with C++ for computation algorithms.

3. Interoperability: Swift is designed to coexist with the Objective-C language, allowing you to incorporate Objective-C files into Swift projects and vice versa.

4. Memory Management: Swift uses Automatic Reference Counting (ARC) for memory management, making it easier for developers.

Swift Basics: Variables, Constants, and Types

Variables

Variables are named storage that our programs can manipulate. They are declared in Swift using the var keyword. For instance:

var age: Int = 25

Constants

Constants are similar to variables, except the value cannot be changed once it's set. They are declared using the let keyword. For example:

let pi: Double = 3.14159

Data Types

Swift has several basic data types:

1. Integers: Integers are whole numbers with no fractional component, such as 42 and -23. You declare integers with the Int keyword. For instance:

var number: Int = 10

2. Doubles and Floats: These are used for floating-point numbers, or numbers with a fractional component. Double has a precision of at least 15 decimal digits, whereas the precision of Float can be as little as 6 decimal digits. For example:

var pi: Double = 3.14159

3. Booleans: Boolean values are referred to as logical because they can only ever be true or false. Swift uses the Bool type for boolean values. For instance:

var isActive: Bool = true

4. Strings: Strings are sequences of characters, such as "hello, world". They are represented by the String type. For example:

var greeting: String = "Hello, world!"

Swift also supports complex data types like arrays, dictionaries, optionals, tuples, and many more which we'll explore in the upcoming classes.
