Control Flow in Swift

Control flow refers to the order in which the program's code executes. The control flow statements are used to repeat tasks, select different code blocks to execute, and transfer the program's control flow.

In Swift, we have several control flow statements:

1. Conditional Statements: These include if, else if, and else statements used to execute different code blocks based on certain conditions.

Example:


var temperature = 30
if temperature < 20 {
  print("It's cold!")
} else if temperature > 30 {
  print("It's hot!")
} else {
  print("It's pleasant!")
}

2. Switch Statement: The switch statement is a powerful tool for control flow that can be used instead of if-else-if chains.

Example:


var fruit = "apple"
switch fruit {
  case "banana":
      print("I like bananas.")
  case "apple":
      print("Apples are tasty.")
  default:
      print("All fruits are delicious.")
}

3. Loop Statements: Loops are used to perform a set of tasks until a certain condition is met. Swift has for, while, and repeat-while loops.

Example:


for i in 1...5 {
  print(i)
}

Introduction to Functions in Swift

Functions are self-contained blocks of code that perform a specific task. Swiftâ€™s unified function syntax is flexible and expressive.

Functions in Swift are declared using the func keyword. For example:

Example:


func greet(person: String) -> String {
  let greeting = "Hello, " + person + "!"
  return greeting
}

In this example, greet is a function that takes a parameter person of type String and returns a String. To call this function, you would use greet("John").

Functions can have multiple parameters, and they can also return multiple values using tuples. Swift also provides advanced features such as nested functions, variadic parameters, and closure expressions, which we will cover in upcoming classes.
