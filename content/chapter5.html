Class 5 - Basic iOS Development

Introduction to Xcode and Interface Builder

Xcode is the Integrated Development Environment (IDE) provided by Apple for developers to create apps for its various platforms including iOS, macOS, tvOS, and watchOS.

Xcode provides a suite of tools including a source editor, interface builder, asset catalog, and many more. It also includes the iOS Simulator, where you can test your app without needing an actual device.

Interface Builder is a software development application in Xcode's suite that allows developers to create graphical user interfaces for their apps. The Interface Builder allows you to create and design your app's UI visually, without having to write all the code by hand.

Creating Your First iOS App

In this section, we'll create a simple "Hello, World!" iOS app.

  1. Create a new project: Open Xcode and choose "Create a new Xcode project". Then, select "App" under the iOS tab, and click "Next".
  2. Setup your project: Give your project a name (e.g., "HelloWorld") and make sure that "Swift" is selected as the language. Click "Next" and choose a location to save your project.
  3. Understand the project structure: In the Project Navigator on the left, you'll see several files and folders. The two most important files for now are SceneDelegate.swift and ViewController.swift. The SceneDelegate.swift is the entry point of your app, and ViewController.swift contains the main logic for your app's main screen.
  4. Design the interface: Go to Main.storyboard. Here you can design the user interface for your app. Drag a label from the Object Library onto the view and place it at the center of the screen. Change the text of the label to "Hello, World!".
  5. Run your app: To run your app, select the appropriate simulator from the toolbar (e.g., iPhone 11), and then click the "Play" button in the top left corner. You should see your "Hello, World!" message appear on the screen.

Congratulations on building your first iOS app!
