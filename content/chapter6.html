Class 6 - Working with UI Elements

Introduction to UI Elements: Labels, Buttons, Text Fields

User Interface (UI) elements are the parts we use to build apps or websites. These could be input controls, navigational components, informational components, and containers.

In iOS, some of the most frequently used UI elements are:

  1. Labels (UILabel): A label is a static text field which can be used to display a single line or multiple lines of text. Labels do not allow user interaction.
  2. Buttons (UIButton): Buttons are fundamental UI elements in any graphical user interface. They provide a simple way for users to interact with the interface. In response to a tap, buttons can trigger a variety of actions.
  3. Text Fields (UITextField): Text fields allow users to input text data with the keyboard. They are often used to gather small amounts of text, such as an email address, password, etc.

Building a Basic UI for an iOS App

In this section, we'll create a basic UI containing a label, a text field, and a button. When the button is pressed, we'll update the text of the label to reflect what's in the text field.

  1. Creating UI Elements: Open your project and navigate to the Main.storyboard. Drag a label, a text field, and a button from the Object Library onto your view controller.
  2. Connecting UI Elements to Code: After designing the layout, we need to connect these UI elements to our code. To do this, open Main.storyboard and ViewController.swift side by side. Control-drag from each UI element to the ViewController.swift file and create a new outlet for each one.
  3. Writing the Interaction Logic: The next step is to create an action that gets triggered when the button is tapped. Control-drag from the button in the storyboard to the ViewController.swift file and create an action this time. Inside this action, update the text property of the label to reflect the text field's contents.

    Example Code:

    @IBOutlet weak var myLabel: UILabel!
@IBOutlet weak var myTextField: UITextField!

@IBAction func myButtonPressed(_ sender: UIButton) {
  myLabel.text = myTextField.text
}
  4. Run your app: Select the appropriate simulator and then click the "Play" button. Now when you enter some text into the text field and press the button, the label's text should update.
